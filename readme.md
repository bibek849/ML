# Product Sales ML Analysis

This project implements:
- Data preprocessing (cleaning, outlier capping, normalization)
- K-means clustering from scratch (NumPy)
- Regression models to predict product profit
- A Streamlit dashboard for non-technical users

## Files

- `app.py` – Streamlit app with full analysis + UI
- `product_sales.csv` – dataset
- `requirements.txt` – Python dependencies

## How to run locally

```bash
pip install -r requirements.txt
streamlit run app.py
